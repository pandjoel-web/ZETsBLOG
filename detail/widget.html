  • Structural Widget â€“ seperti Container, Column, Row

Stateless vs Stateful Widget

Widget dibagi menjadi dua jenis utama:

  • StatelessWidget â†’ Tidak menyimpan state (kondisi berubah)
  • StatefulWidget â†’ Menyimpan dan merespons perubahan state

Contoh StatelessWidget


class MyTextWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Text("Hello Flutter!");
  }
}

Contoh StatefulWidget


class CounterWidget extends StatefulWidget {
  @override
  _CounterWidgetState createState() => _CounterWidgetState();
}

class _CounterWidgetState extends State<CounterWidget> {
  int counter = 0;

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text("Count: \$counter"),
        ElevatedButton(
          onPressed: () {
            setState(() {
              counter++;
            });
          },
          child: Text("Tambah"),
        )
      ],
    );
  }
}

Widget Layout Populer

  • Row & Column: Layout horizontal dan vertikal
  • Container: Untuk membungkus dan menstyling widget lain
  • Expanded: Untuk mengisi ruang kosong di Row/Column
  • Stack: Untuk overlay/memposisikan widget bertumpuk

Contoh Layout Sederhana


Widget build(BuildContext context) {
  return Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: [
      Text("Selamat datang!"),
      SizedBox(height: 20),
      ElevatedButton(
        onPressed: () {},
        child: Text("Mulai"),
      ),
    ],
  );
}

Tips Menggunakan Widget

  • Gunakan widget kecil agar mudah dipelihara
  • Pisahkan layout ke file/class berbeda bila kompleks
  • Gunakan padding, margin, dan alignment dengan konsisten
  • Manfaatkan widget Builder dan MediaQuery untuk responsif

Kesimpulan

Widget adalah pondasi utama dalam Flutter. Dengan memahami cara kerja dan jenis-jenis widget, kamu bisa membuat UI yang powerful dan fleksibel. Teruslah bereksperimen dan manfaatkan dokumentasi resmi Flutter!
