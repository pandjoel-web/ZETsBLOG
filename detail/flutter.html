Belajar Flutter: Framework Mobile Masa Kini
Panduan dasar untuk mulai ngoding aplikasi Android & iOS dengan satu codebase

📱 Mobile Development 🛠️ Framework: Flutter 👨‍💻 Bahasa: Dart

Apa itu Flutter?

Flutter adalah framework open-source buatan Google untuk membangun aplikasi mobile, web, dan desktop dari satu codebase. Flutter menggunakan bahasa pemrograman Dart dan memiliki engine rendering sendiri, sehingga UI-nya bisa konsisten di berbagai platform.

Kenapa Harus Pakai Flutter?

  • 🚀 Cepat & Produktif: Fitur Hot Reload memungkinkan pengembang melihat perubahan secara langsung.
  • 📱 UI Konsisten: Tidak bergantung ke komponen native, jadi desain bisa presisi sesuai keinginan.
  • 🔁 Reuse Codebase: Satu kode bisa dijalankan di Android, iOS, Web, bahkan Desktop.
  • 💼 Backed by Google: Didukung dan dikembangkan langsung oleh Google, dengan komunitas yang besar.

Struktur Dasar Proyek Flutter

  • /lib: Tempat kode Dart utama. Biasanya dimulai dari main.dart
  • /assets: Folder untuk menyimpan gambar, font, dsb.
  • pubspec.yaml: File konfigurasi untuk dependencies dan aset

Contoh Kode Flutter


import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Halo Flutter',
      home: Scaffold(
        appBar: AppBar(title: Text('Halo Flutter')),
        body: Center(child: Text('Hello, World!')),
      ),
    );
  }
}

Widget di Flutter

Semua elemen di Flutter adalah widget: dari Text, Button, sampai Layout seperti Column dan Row.

  • StatelessWidget: Widget yang tidak berubah saat app berjalan.
  • StatefulWidget: Widget yang bisa berubah (misal: toggle switch, counter).
  • Material Widgets: Komponen UI standar Android
  • Cupertino Widgets: Komponen UI ala iOS

Installasi Flutter

  1. 1. Download Flutter SDK dari [flutter.dev](https://flutter.dev)
  2. 2. Tambahkan ke PATH agar bisa akses dari terminal
  3. 3. Jalankan flutter doctor untuk cek environment
  4. 4. Buat project dengan flutter create namaprojek
  5. 5. Jalankan app pakai flutter run

Tips Pengembangan Flutter

  • 🧪 Selalu gunakan Hot Reload saat pengembangan
  • 🎯 Gunakan State Management (Provider, Riverpod, Bloc)
  • 🎨 Rancang UI dulu di Figma sebelum coding
  • 🔍 Pisahkan widget ke file terpisah untuk maintainabilitas

Kesimpulan

Flutter cocok banget buat kamu yang pengen bikin aplikasi keren tapi efisien. Dengan satu codebase dan komunitas yang aktif, Flutter bisa jadi pilihan utama dalam pengembangan mobile modern.
