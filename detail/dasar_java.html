Dasar Bahasa Pemrograman Java
Pemahaman dasar untuk mulai ngoding dengan Java

🧠 Bahasa: Java 🖥️ Paradigma: OOP 🌐 Platform: JVM

Apa Itu Java?

Java adalah bahasa pemrograman berorientasi objek (OOP) yang dikembangkan oleh Sun Microsystems (sekarang dimiliki oleh Oracle). Java terkenal dengan prinsip "tulis sekali, jalankan di mana saja" karena berjalan di atas JVM (Java Virtual Machine).

Struktur Program Dasar Java


public class Main {
    public static void main(String[] args) {
        System.out.println("Halo, dunia!");
    }
}
  • public class → mendefinisikan kelas utama
  • main() → titik awal eksekusi program
  • System.out.println() → mencetak ke layar

Tipe Data Dasar

  • int → bilangan bulat
  • double → bilangan desimal
  • char → karakter tunggal
  • String → teks
  • boolean → true atau false

Contoh Deklarasi Variabel


int umur = 20;
String nama = "Zet";
double tinggi = 175.5;
boolean isActive = true;

Kontrol Alur

1. Percabangan (if/else)


if (umur >= 18) {
    System.out.println("Dewasa");
} else {
    System.out.println("Anak-anak");
}

2. Perulangan (loop)


// for loop
for (int i = 1; i <= 5; i++) {
    System.out.println("Ke - " + i);
}

Array dan ArrayList


// Array
int[] angka = {1, 2, 3};

// ArrayList
import java.util.ArrayList;
ArrayList<String> namaSiswa = new ArrayList<>();
namaSiswa.add("Ayu");
namaSiswa.add("Budi");

Fungsi / Method


public static int tambah(int a, int b) {
    return a + b;
}

Konsep OOP di Java

Java mendukung 4 pilar OOP: Encapsulation, Abstraction, Inheritance, Polymorphism.

Contoh Class dan Object


public class Mahasiswa {
    String nama;
    int umur;

    void perkenalan() {
        System.out.println("Halo, saya " + nama + ", umur " + umur + " tahun.");
    }
}

public class Main {
    public static void main(String[] args) {
        Mahasiswa mhs = new Mahasiswa();
        mhs.nama = "Gesya";
        mhs.umur = 21;
        mhs.perkenalan();
    }
}

Kesimpulan

Java adalah bahasa yang andal dan banyak digunakan di industri. Setelah menguasai dasar-dasarnya, kamu bisa lanjut ke topik seperti GUI, database, hingga Android development.
