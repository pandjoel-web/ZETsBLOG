Dasar CSS untuk Pemula

🗂️ Kategori: Web Development 🗓️ Dipublikasikan: 17 Juli 2025

CSS (Cascading Style Sheets) adalah teknologi penting dalam pengembangan web yang digunakan untuk mengatur tampilan elemen HTML di halaman web. Dengan CSS, kita bisa mengubah warna, ukuran, layout, hingga animasi pada website.

Apa Itu CSS?

CSS merupakan bahasa desain yang bertugas mempercantik halaman web. Jika HTML adalah kerangka, maka CSS adalah gaya dan pakaian yang dikenakan oleh halaman tersebut.

"CSS membuat halaman web tidak hanya berfungsi, tetapi juga indah dipandang."

Tanpa CSS, halaman web hanya akan menampilkan teks hitam di latar putih, tanpa warna, tanpa tata letak yang menarik.

Cara Menyisipkan CSS

Ada tiga cara umum untuk menggunakan CSS:

  1. Inline CSS – langsung di dalam tag HTML
  2. Internal CSS – di dalam tag <style> di bagian <head>
  3. External CSS – melalui file .css terpisah

Contoh Internal CSS

<style>
  body {
    background-color: #f0f0f0;
    color: #333;
  }
</style>

Selektor dan Properti

CSS bekerja dengan selektor dan properti. Contohnya: 

p {
  font-size: 16px;
  color: blue;
}

Kode di atas akan membuat semua paragraf (<p>) memiliki ukuran teks 16 piksel dan berwarna biru.

Penutup

Menguasai CSS akan membuka pintu ke dunia desain web yang lebih dinamis dan menarik. Setelah memahami dasarnya, kamu bisa lanjut ke Flexbox, Grid, dan bahkan animasi menggunakan CSS!
