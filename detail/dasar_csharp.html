Apa Itu C#?

C# (dibaca: "C-sharp") adalah bahasa pemrograman berorientasi objek (OOP) yang dikembangkan oleh Microsoft. Bahasa ini berjalan di atas framework .NET dan sangat populer untuk membangun aplikasi desktop, web, game (Unity), dan banyak lagi.

Struktur Program Dasar C#


using System;

class Program {
    static void Main(string[] args) {
        Console.WriteLine("Hello, world!");
    }
}
  • using System; ‚Üí mengimpor namespace
  • Main ‚Üí method utama yang dijalankan pertama kali
  • Console.WriteLine() ‚Üí untuk mencetak output ke layar

Tipe Data Dasar

  • int ‚Üí bilangan bulat
  • double ‚Üí bilangan pecahan
  • char ‚Üí satu karakter
  • string ‚Üí teks
  • bool ‚Üí nilai true/false

Contoh Deklarasi Variabel


int umur = 20;
string nama = "Zet";
double tinggi = 175.5;
bool isActive = true;

Kontrol Alur

1. Percabangan (if/else)


if (umur >= 18) {
    Console.WriteLine("Dewasa");
} else {
    Console.WriteLine("Anak-anak");
}

2. Perulangan (loop)


// for loop
for (int i = 1; i <= 5; i++) {
    Console.WriteLine("Ke - " + i);
}

Array dan List


// Array
int[] angka = { 1, 2, 3 };

// List
List<string> namaSiswa = new List<string>() { "Ayu", "Budi" };
namaSiswa.Add("Cinta");

Fungsi / Method


static int Tambah(int a, int b) {
    return a + b;
}

Konsep OOP di C#

C# mendukung 4 pilar OOP: Encapsulation, Abstraction, Inheritance, Polymorphism.

Contoh Class dan Object


class Mahasiswa {
    public string Nama;
    public int Umur;

    public void Perkenalan() {
        Console.WriteLine($"Halo, saya {Nama}, umur {Umur} tahun.");
    }
}

class Program {
    static void Main() {
        Mahasiswa mhs = new Mahasiswa();
        mhs.Nama = "Gesya";
        mhs.Umur = 21;
        mhs.Perkenalan();
    }
}

Kesimpulan

C# adalah bahasa pemrograman yang kuat dan fleksibel untuk berbagai jenis aplikasi. Dengan dasar-dasar ini, kamu bisa lanjut belajar GUI, database, hingga membangun game dengan Unity.
