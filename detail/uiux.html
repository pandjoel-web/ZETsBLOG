Langkah Mendesain Aplikasi Flutter di Figma

  1. Buat Frame Sesuai Ukuran
    Gunakan frame mobile seperti 360x800 (Android) atau iPhone 14 Pro. Figma sudah menyediakan presetnya.
  2. Gunakan Grid dan Auto Layout
    Supaya layout tetap rapi dan responsif ke berbagai ukuran device.
  3. Gunakan Style Consistent
    Buat text style, color style, dan component agar mudah di-maintain dan cocok untuk Flutter UI reuse.
  4. Simulasikan Alur UX
    Hubungkan antar frame jadi prototype interaktif, tes flow sebelum development.
  5. Export Asset (SVG/PNG)
    Gambar ikon atau ilustrasi bisa diexport langsung untuk digunakan di Flutter dengan `Image.asset()` atau `SvgPicture.asset()`.
